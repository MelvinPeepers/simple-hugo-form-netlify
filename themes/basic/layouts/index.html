Simple Netlify Form in Hugo

What you need to add to create a simple Netlify form

  • Add either data-netlify="true"> or netlify attribute.
  • Each form should have a different name attribute.
  • The basic code can be viewed here. The code needs to be added to the index.html

Add the site to Netlify

  • From the Site page, click 'New site from Git' button.
  • Click GitHub under "Connect to Git provider".
  • Search for the repo with the form or select the repo with the form.
  • Stick with default build settings and click 'Deploy Site'.

If forms are detected.

  • You'll see on the Deploy log under 'Netlify Build Complete'
  • Processing form - form name> and Detected form fields:
  • Under the Forms tab, you'll see 'Active forms' with the form name

Testing forms

  • Go to the live site
  • Enter information then click submit
  • You should see a 'Thank you!'
  • Going back to the Forms tab, click on the form name under 'Active forms' and you should then see the form that was submitted.
